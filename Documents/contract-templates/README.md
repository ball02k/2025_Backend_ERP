# Contract Templates

This directory contains DOCX templates used for generating contracts in the ERP system.

## Templates

- **base.docx**: Default contract template used when creating new contracts

## Usage

Templates are automatically copied and used when:
1. Creating a new contract via Direct Award
2. Opening a contract in the ONLYOFFICE editor for the first time

## Customization

You can customize the `base.docx` template by:
1. Opening it in Microsoft Word or LibreOffice
2. Editing the content, formatting, and placeholders
3. Saving the changes

The system will use the updated template for all new contracts.

## Placeholders

Recommended placeholders to include in your templates:
- `[DATE]` - Contract date
- `[CONTRACTOR_NAME]` - Contractor/supplier name
- `[CLIENT_NAME]` - Client/project name
- `[CONTRACT_VALUE]` - Total contract value
- `[TERMS]` - Terms and conditions
